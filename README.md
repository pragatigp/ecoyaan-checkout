# Ecoyaan Checkout Flow

This project is a simplified checkout flow built using Next.js.

## Features
- Cart page with product list
- Server-Side Rendering using getServerSideProps
- Shipping form with validation
- Payment confirmation screen
- Order success page

## Tech Stack
- Next.js
- React
- Tailwind CSS

## Project Structure
components/
context/
pages/
pages/api/cart.js

## Run Locally
npm install
npm run dev

Then open:
http://localhost:3000/cart
